# sample_data/

Put your watch history file in this folder before running the analysis.

Accepted file names:
- `watch-history.json` — from Google Takeout (recommended)
- `watch-history.html` — from Google Takeout (alternative)
- any `.pdf` — saved from the YouTube History page

These file types are listed in `.gitignore` and will NOT be committed to GitHub.
Your data stays on your computer only.
